P tag: The paragraph tag allows you to write a paragraph in your HTML file.

H1 tag: Allows you to define the most important HTML headings.

A tag: Allows you to add a link in your HTML file.

Img tag: Allows you to add a image in your HTML file.

Ul tag: Allows you to add a description list in your HTML file.
